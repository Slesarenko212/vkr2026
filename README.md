# Гибридная нейросетевая модель обеспечения ИБ IT-инфраструктуры

Данный репозиторий содержит программную реализацию системы обнаружения вторжений (СОВ) нового поколения для Центра мониторинга SOC. В основе системы лежит гибридная архитектура **CNN-LSTM**, позволяющая осуществлять двухфакторный (пространственно-временной) анализ сетевой телеметрии.

## 🚀 Научная новизна исследования
1. **Двухфакторный анализ**: Использование одномерных сверточных слоев (Conv1D) для автоматического извлечения признаков заголовков пакетов совместно с блоками LSTM для выявления распределенных и затяжных во времени атак (APT, Slow DoS).
2. **Адаптивное реагирование**: Логический контроллер распределения рисков на основе динамических порогов отсечения вердиктов модели.

## 📁 Структура репозитория
* `src/model.py` — Сборка гибридного ядра CNN-LSTM средствами TensorFlow/Keras с использованием механизмов `TimeDistributed`.
* `src/utils.py` — Модуль предобработки сетевых данных: гигиеническая фильтрация, линейная нормализация Min-Max, оверсемплинг миноритарных классов (SMOTE).
* `src/pipeline.py` — Реализация алгоритма скользящего окна (`Sliding Window`) для генерации трехмерных тензоров $[M, T, F]$.
* `src/controller.py` — Двухуровневый `ResponseController` для генерации управляющих команд (Active Response / Honeypot / SIEM Log).
* `tests/load_testing.py` — Скрипт нагрузочного тестирования операционных показателей системы (EPS и Latency).

## 📊 Целевые метрики эффективности
* Точность обнаружения аномалий (Precision): **>= 90%**
* Уровень ложных срабатываний (False Positive): **<= 5%**
* Скорость обработки входящего потока данных: **>= 500 событий/сек (EPS)**
